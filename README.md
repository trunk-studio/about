# 創科資訊股份有限公司

## 公司簡介

創科資訊股份有限公司創立於 2015 年，是一家年輕、充滿活力的新創公司，成立目的在於提供客戶高品質的軟體設計服務，不只是擁有專業技術，我們更有親切、誠懇的服務態度，完整考量客戶及使用者的需求共創多贏。

## 服務內容

AWS 雲端應用開發、響應式網頁設計、客製化軟體設計、企業管理系統開發、數位內容平台、軟體開發培訓
